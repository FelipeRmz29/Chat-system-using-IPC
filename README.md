# Chat-system-using-IPC

A multi-client chat system using Inter-Process Communication (IPC) mechanisms.

## Features
- Multi-client server architecture.
- Broadcast and private messages.
- List online users.
- Graceful disconnection handling.

## Requirements
- Linux OS
- GCC compiler
- POSIX libraries
