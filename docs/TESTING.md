# Testing Scenarios

## Test 1: Basic Communication
1. Start the server.
2. Start two clients.
3. Send messages between clients.

## Test 2: Multiple Clients
1. Start the server.
2. Start 10 clients using a script.
3. Verify all clients can communicate.

## Test 3: Disconnections
1. Start the server and 3 clients.
2. Kill one client abruptly.
3. Verify the server detects and cleans up the disconnected client.
